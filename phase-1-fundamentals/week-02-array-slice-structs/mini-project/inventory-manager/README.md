# Inventory Manager CLI

A simple CLI tool to manage store inventory.

Go concepts:
- Structs with methods
- Maps and slices for data storage
- Pointer receivers for mutation
- Interactive menu with switch
- Basic validation and error messages
- Input handling with bufio.Reader
- defer for cleanup message


## Features
- Add new items with name, price, category, and initial quantity
- Increase or decrease stock of existing items
- List all items with details: price, quantity, and total value
Calculate total inventory value
Delete items
Case-insensitive item names
- Simple interactive menu using loops, functions, structs, pointers, and maps
- Cannot reduce stock below zero.
